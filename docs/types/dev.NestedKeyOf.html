    Recursively creates a union of property paths for nested objects. Limited to 3 levels of depth to prevent infinite type recursion. Excludes function properties from the path generation.

    Type Parameters

    • ObjectType

      The object type to extract property paths from

    • Depth extends number = 3

      Internal counter to limit recursion depth (default: 3)

    Example

    type Paths = NestedKeyOf<{ a: { b: { c: number } } }>
    // Result: "a" | "a.b" | "a.b.c"
